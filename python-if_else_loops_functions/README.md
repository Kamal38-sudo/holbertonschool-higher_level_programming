# Random Number Checker

Bu layihə Python proqramıdır və təsadüfi ədəd yaradır, sonra həmin ədədin **müsbət**, **sıfır** və ya **mənfi** olduğunu yoxlayır.

## Tələblər

- Python 3.x
- Standart `random` modulu (Python ilə gəlir, əlavə quraşdırmaya ehtiyac yoxdur)

## İstifadə qaydası

1. Layihəni klonlayın və ya `random_number_checker.py` faylını açın.
2. Terminalda aşağıdakı əmri işlədin:

```bash
python random_number_checker.py
